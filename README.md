# **Inventory Stock Management System**

### **Description**

The **Inventory Stock Management System** is a comprehensive solution built with **Next.js** to efficiently manage product inventories, track stock levels, monitor order statuses, and generate detailed reports. It is designed for businesses that need to maintain control over their inventory, suppliers, and sales data in real-time. The system features an intuitive admin dashboard, powerful reporting tools, and the ability to manage inventory from stock overview to product categorization.

## **Installation**

Follow these steps to set up and run the project locally:

1. **Clone the repository**:

   ```bash
   git clone https://github.com/niroberto/inventory_management_system.git
   cd inventory_management_system
   ```

2. **Install dependencies**:

   ```bash
   npm install
   ```

3. **No need Set up environment variables**:



4. **Run the development server**:

   ```bash
   npm run dev
   ```

   Visit [http://localhost:3000](http://localhost:3000) to view the app in your browser.

## **Features**

### **1. Inventory Management**

- Add, edit, and delete products.
- Track stock levels, product descriptions, pricing, and inventory status (in stock, out of stock).
- Categorize products into various groups for easier management.

### **2. Supplier Management**

- Maintain a list of suppliers, contact details, and product stock they provide.
- Track orders from each supplier with order status updates (pending, received, shipped).

### **3. Order Management**

- Manage sales orders with real-time tracking of product availability and order status.
- View detailed order information including customer details, purchased items, and shipping status.

### **4. Reports & Analytics**

- Generate detailed reports on product performance, stock levels, and sales activity.
- View historical data to analyze trends in product demand and supplier performance.

### **5. Role-based Access Control**

- Different user roles (Admin, Manager, etc.) can access specific sections of the system.
- Admins can manage users, inventory, and reports while managers can handle orders and suppliers.

### **6. Real-time Notifications**

- Get notified about low stock levels, incoming orders, and updates on product shipments.

## **Usage**

### **Frontend (Next.js)**

1. **Dashboard**: The main interface that provides an overview of stock levels, product details, and order statuses.
2. **Product Management**: Admins can add, edit, or remove products, and update stock quantities.
3. **Supplier Management**: View supplier information and manage orders placed with them.
4. **Order Management**: Track sales orders, process returns, and update order statuses.
5. **Reports**: Generate and export reports related to stock movements, sales data, and inventory status.
